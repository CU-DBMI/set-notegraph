The usual suspects. Here's a good overview, https://ploomber.io/blog/survey/. The ploober document lists the following, but it's also a sales site, so listing tools separately without opinion:

  • Ploomber
  • Airflow
  • Dagster
  • DVC (Data Pipelines)
  • Elyra
  • Flyte
  • Kale
  • Kedro
  • Kubeflow pipelines
  • Luigi
  • Metaflow
  • Prefect
  • Tensorflow Extended (TFX)

DBT is an "data transformation tool," more focused on flowing SQL and database data, but very powerful there.

Nextflow, a Java tool, is especially liked in the academic research community.

Useful in the academic and research environment, we've run across SnakeMake.

Ray Workflows - Part of the Ray distributed compute project.

Globus Flows - Part of Globus, a research-focused data management suite.
