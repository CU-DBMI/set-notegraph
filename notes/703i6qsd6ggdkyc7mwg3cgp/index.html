Validation

Tools for data validation.

Schema-based Validation

Using static schema definitions to validate data.

Apache-project Based

  • Avro (pythonic implementation): Python library for using Apache Avro.
  • Apache Arrow Table Schema Validate (PyArrow): Validates Apache Arrow schema.
  • Apache Parquet Schema (PyArrow): Gathers but no embedded validate functionality for Parquet-based schema. Note: may use Arrow-based schema validation through PyArrow.

JSON-based

  • jsonschema (pythonic implementation): Python library for using JSON Schema functionality.

Independent

  • Cerberus: custom data validation system for python.
  • Deequ: schema and profile based data unit-testing framework based on Apache Spark.
  • Pydantic: Pythonic data validation library.
  • Marshmallow: Pythonic data ORM/ODM and validation library.
  • Pandas assert_frame_equal: validate Pandas dataframes against one another.
  • Pandera: Pythonic dataframe-focused schema validation.
  • Voluptuous: custom data validation system for python.
  • Schema (Python): Pythonic schema validation.

Language

  • Cuelang: data, schema, and validation language, all-in-one.

Profile-based Validation

Using profiles of data (as opposed to static schema definition) to validate data.

  • Great Expectations: implements profile-based data quality checks and validation for your data.
  • Soda: implements profile-based data quality checks and validation for your data.
  • DeepChecks: intended from ML data profile validation.
